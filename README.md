# ASLConcorsiSmartCopier
Script per trasferire le informazioni fondamentali da una piattaforma Concorsi Smart all'altra, evitando di dover copiare manualmente tutti i campi ogni volta che ci si iscrive ad un concorso ASL diverso. Include al momento: Titoli, Esperienze precedenti nella PA, Articoli e Pubblicazioni e Corsi.
